# BookRev

A book review web application where users can browse and post book reviews, with the option to listen to music via Spotify while they read.

---

## About

BookRev is a full-stack web app built around the idea that reading and music go hand in hand. Users can explore reviews, contribute and optionally queue up music through a Spotify integration, all without leaving the page.

## Tech Stack

- **Frontend:** HTML, CSS, EJS 
- **Backend:** Node.js
- **Database:** MongoDB
- **API:** Spotify Web API

## Features

- Browse book reviews posted
- Post your own book reviews
- Listen to music via Spotify integration while on the page
- Session-based data handling with MongoDB

## Notes

- Built in 2019/2020. Some dependencies or API behaviour may have changed since then, in particular, Spotify's API and OAuth flow have evolved, so the music feature may require updates to work today.
- MongoDB stores data entered during the session.
